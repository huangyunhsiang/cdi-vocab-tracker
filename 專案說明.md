# 專案說明

## 專案目的
**寶寶成長追蹤** PWA（原名 CDI 詞彙追蹤，2026-07-07 擴充改名；repo 名／網址仍 cdi-vocab-tracker）。兩大模組：①**詞彙**（CDI 公開類別架構）三軌記錄（理解／表達／手勢）、成長曲線＋詞彙爆發偵測、類別分佈、研究級 CSV 匯出；②**動作里程碑**（WHO 六大粗大動作）記錄達成日＋WHO 正常窗口對照。給使用者與恬恬老師共同記錄自家孩子（2025-07-21 生）。Phase 2 預留 LINE 快速記錄入口。

## 技術棧 / 啟動方式
- vanilla JS PWA（無 build step）＋ Chart.js CDN；紙感極簡主題
- 資料層可插拔：預設 localStorage；`js\firebase-config.js`（gitignore，樣板為 `firebase-config.sample.js`）填入設定後啟用 Firestore 同步
- 本機啟動：`python -m http.server 8770`
- 測試：`node --test tests\`（analytics 純函式）

## 版權紅線
MCDI-T 正式題本有版權，**任何情況都不得把題本詞項寫進 repo**；題本走使用者本機檔案載入（`js\wordlist-loader.js`）。
